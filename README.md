## ‚ö° Getting Started

Want to run GrooveWave locally?

1. **Clone the repository**
   ```bash
   git clone https://github.com/bhanu2006-24/groovewave.git
   cd groovewave
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
